# Python_projects
Python scripts
In this mini project, the game is developed  where the user plays against the computer. The project demonstrates the use of programming concepts such as variables, conditional statements, iterative statements, random choice generation, and user interaction.

The main objectives of the Rock Paper Scissors mini project are:
  •	To develop a simple interactive game using programming concepts
  •	To understand the use of conditional statements, iterative statements,. Etc
  •	To implement random choice generation using random module.

The mini project named ROCK, PAPER, SCISSORS GAME is developed using following like Programming Language: Python (Python 3.9.6) and IDE: Visual Studio Code (1.122.1 (Universal))

The game works by allowing the user player to choose one option among Rock, Paper, or Scissors using input function  . The computer also randomly selects one option using random module. Based on predefined game rules like following,
  •	If both player and computer choices are the same like (Rock-Rock, Scissor-Scissor and Paper-Paper), the round is a tie and both scores are same. 
  •	If the real player’s choice beats the computer’s choice according to the rules, the player wins and the player’s score is increased.
  •	Otherwise, the computer wins and its score is increased.

Likewise program compares both choices, tracks rounds, scores and declares the winner. The game can be played repeatedly, and scores are maintained throughout the session and it display the rounds, scores and winners.
